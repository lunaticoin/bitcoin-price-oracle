Bitcoin Price Oracle API serves BTC/USD prices computed 100% on-chain from your own Bitcoin node. Data is provided by the brk_oracle algorithm, which detects round-dollar transaction patterns in block outputs. No external APIs or exchange data are used. The API runs on port 3200 by default.

Endpoints

GET /api/price/latest
Returns the price at the most recently synced block.
curl http://umbrel.local:3200/api/price/latest { "height": 942928, "price_usd": 68260.19, "timestamp": 1774874850 }
GET /api/price/{height}
Returns the price at a specific block height.
curl http://umbrel.local:3200/api/price/800000 { "height": 800000, "price_usd": 29806.73, "timestamp": 1690168629 }
GET /api/price/date/{YYYY-MM-DD}
Returns the price at the first block on or after the given UTC date.
curl http://umbrel.local:3200/api/price/date/2024-01-15 { "date": "2024-01-15", "height": 825000, "price_usd": 42853.0, "timestamp": 1705276800 }
GET /api/price/range?from={}&to={}
Returns prices for a range of block heights or dates. Max 10,000 entries per request. The from and to parameters accept either block heights (numeric) or dates (YYYY-MM-DD format).
curl "http://umbrel.local:3200/api/price/range?from=800000&to=800010" [ {"height":800000,"price_usd":29806.73,"timestamp":1690168629}, {"height":800001,"price_usd":29810.45,"timestamp":1690169234}, ... ]
GET /health
Returns sync status and progress information.
curl http://umbrel.local:3200/health { "synced_height": 942928, "chain_tip": 942930, "syncing": false, "progress_percent": 99.99 }

Configuration

Port: 3200 (default)
Data source: brk_oracle on-chain algorithm
Accuracy: 0.11% median error (from block 550,000 / Nov 2018)
Resolution: Per-block (~10 min)
Coverage: Block 0 to chain tip (embedded prices for blocks 0-549,999, oracle-computed from 550,000+)
Storage: ~12 MB (prices.bin + timestamps.bin)
RAM usage: <100 MB

Integration Examples

JavaScript
const res = await fetch('/api/price/latest'); const { price_usd, height } = await res.json(); console.log(`BTC: $${price_usd} at block ${height}`);
Python
import requests r = requests.get('http://umbrel.local:3200/api/price/latest') print(f"BTC: ${r.json()['price_usd']}")
cURL
# Latest price curl -s http://umbrel.local:3200/api/price/latest | jq .price_usd # Price on a specific date curl -s http://umbrel.local:3200/api/price/date/2024-06-15 | jq .
