The Grimoire (API Documentation) — these endpoints serve BTC/USD prices divined entirely from the blockchain, via the brk_oracle algorithm. No exchange feeds. No external APIs. Pure on-chain revelation. Port 3200 by default.
GET/api/price/latest
Returns the price at the most recently synced block.
curl http://umbrel.local:3200/api/price/latest {"height":942928,"price_usd":68260.19,"timestamp":1774874850}
GET/api/price/{height}
Returns the price at a specific block height.
curl http://umbrel.local:3200/api/price/800000 {"height":800000,"price_usd":29806.73,"timestamp":1690168629}
GET/api/price/date/{YYYY-MM-DD}
Returns the price at the first block on or after the given UTC date.
curl http://umbrel.local:3200/api/price/date/2024-01-15 {"date":"2024-01-15","height":825000,"price_usd":42853.0,"timestamp":1705276800}
GET/api/price/chart?points={N}
Sampled price history for charting. Returns N evenly-spaced data points across all synced blocks. Default: 600 points. Max: 2000.
curl "http://umbrel.local:3200/api/price/chart?points=100" [{"height":550000,"price_usd":8432.10,"timestamp":...}, ...]
GET/api/price/range?from={}&to={}
Returns prices for a range of block heights or dates (YYYY-MM-DD). Max 10,000 entries per request.
curl "http://umbrel.local:3200/api/price/range?from=800000&to=800010" [{"height":800000,"price_usd":29806.73,"timestamp":1690168629},...]
GET/health
Returns sync status and progress information.
curl http://umbrel.local:3200/health {"synced_height":942928,"chain_tip":942930,"syncing":false,"progress_percent":99.99}
